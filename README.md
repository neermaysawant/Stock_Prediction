# Stock Price Predictor

A Python-based stock price prediction tool that uses Linear Regression to forecast future stock prices for NSE and BSE listed companies.

## Features
- Fetch real-time stock data from Yahoo Finance
- Support for both NSE and BSE listed companies
- Linear Regression based price prediction
- Interactive data visualization
- Model performance evaluation with RÂ² score
- Trading recommendations (Buy/Sell/Hold)
- Customizable historical data period
- Adjustable future prediction timeframe

## Limitations
